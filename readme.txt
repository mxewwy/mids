EN
Hello everybody, I present my mini project on artificial grading based on student's grade point average and Gaussian normal distribution.
In the file gradepoints.txt you need to specify all students and their average points in the format Student:Point.
In the console at startup, you can set the number of marks required to be graded.
RU
Всем привет, представляю мой мини-проект по искусственному выставлению оценок, основанному на среднем балле ученика и нормальном распределении Гаусса.
В файле gradepoints.txt необходимо указать всех учеников и их средние баллы в формате Ученик:Балл.
В консоли при запуске можно выставить количество оценок, требуемых для проставления.
